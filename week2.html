Week 2

Wij hebben deze week gekeken naar het opzetten van een DC-motor circuit om een tape mechanisme aan te sturen, om dit te kunnen vergelijken met een stepper motor circuit.

Ons werd op het hart gedrukt dat deze laatste variant waarschijnlijk een betere keuze zou zijn, maar wij hebben hier nog geen onderdelen voor. Daarom kiezen wij ervoor om de circuits voor beide motoren volledig uit te werken, dan doen wij ervaring op met beide Ã©n kunnen wij een betere vergelijking maken.

Code

De code voor de DC motor was simpel. Deze wordt aangestuurd met pulse width modulation. Wij hebben hiervoor de Teensy_PWM library gebruikt, aangezien die zeer uitgebreid is en meer functies al ingebouwd heeft om preciezer PWM uit te kunnen sturen.

Qua testfunctionaliteiten hebben wij:

  • De motor aan en uit kunnen zetten.
  • De motor op halve snelheid laten draaien, zodat wij dit later kunnen ombuigen naar een potmeter die de snelheid aanpast.

De code is zo geschreven dat deze makkelijk schaalbaar is, dat wil zeggen zonder veel al te veel moeite uitgebreid kan worden naar meerdere, afzonderlijk aan te sturen motoren.

Hardware

Voor het aansturen van de hardware zijn wij gaan samenzitten om de krachten te bundelen. Wij gebruiken een L298N driver, een gebruikelijke keuze. Daarmee is er veel documentatie beschikbaar en was het koppelen goed te doen.

Wij hebben nog een paar kleine aanpassingen gemaakt aan de code zodat deze beter aansluit bij onze doelen, waarbij wij ook gelijk de potmeter hebben aangesloten en code hebben geschreven om de motor variabel heen en weer te kunnen draaien.

Deze code willen wij in de toekomst nog testen met een betere DC-motor om eerlijk te kunnen vergelijken met een Stepper motor.
