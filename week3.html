Voor de software-kant had dit wat meer voeten in de aarde. Wij wilden in eerste instantie gebruikmaken van de TeensyStep library. Deze library heeft hele heldere documentatie, werkt met de Teensy specifiek en is veel uitgebreider dan de standaard Stepper library van Arduino. Het was ons echter niet gelukt om de Arduino IDE de library te doen herkennen. Na een aantal gefrustreerde pogingen hebben we besloten dat het gemak van de library niet opwoog tegen de hoofdpijn.

Daarnaast is de Stepper library van Arduino weliswaar minder uitgebreid, maar wordt deze wel al heel lang door veel mensen gebruikt. De support online is zeer uitgebreid, waarmee alle functionaliteiten van TeensyStep makkelijk na te bootsen zijn.

Deze code heeft dezelfde functies als de code van de DC motor, dat wil zeggen motor aan/uit en halve snelheid. Ook is de code schaalbaar voor meerdere motoren.

Hardware

Voor de hardware kant gebruiken wij een L298N met een NEMA17.
